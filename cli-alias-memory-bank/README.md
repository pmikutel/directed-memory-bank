# memory-bank (reserved)

This npm name is held by the [Directed Memory Bank (DMB)](https://github.com/pmikutel/directed-memory-bank) project.

The actual installer is published as `directed-memory-bank`:

```bash
npx directed-memory-bank init
```

DMB is inspired by the [Cline Memory Bank pattern](https://docs.cline.bot/prompting/cline-memory-bank). Thanks to the Cline team for popularising the idea of structured markdown files for cross-session AI context.
