# Directed Memory Bank (DMB)

**Persistent project understanding for AI coding agents.**

Stop re-explaining your project every AI session. Tool-agnostic project context in plain markdown — works with Claude Code, Cursor, Codex, Gemini, or any agent that reads files.

## What ships at v0.1.0

A small CLI that scaffolds the DMB structure into any project:

```bash
npx directed-memory-bank init
```

…followed by an AI-driven install interview that fills in the structured markdown files (`memory-bank/project/`, `memory-bank/technical/`, `memory-bank/tasks/`) tailored to your project.

The standard itself is just files — readable by any AI tool, version-controlled, no server, no lock-in.
